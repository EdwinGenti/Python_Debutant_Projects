<h1>Python - Divers projets - Niveau Débutant </h1>


<h2>Description</h2>
Le but de ce répertoire est de présenter de nombreux petits projets Python très utiles. Voici les différents projets :
<br>
<br>
1- Extraction des routes et des arrêts de bus dans une ville française au choix (OSMnx), et présententation des couches sur une carte interactive folium. (Terminé)
<br>
2- Calcul de distance GPS
<br>
3- Géocodage d'adresses
<br>
4- Carte interactive avec symbologie personnalisée
<br>
5- Carte choroplèthe classique
<br>
6- Calcul d'aires de polygones
<br>
7- Carte de chaleur
<br>
8- Jointure attributaire / spatiale (Géopandas)
<br>
<br>
<br>
Important : Hormis les projets terminés, tous les autres projets sont des idées qui se développeront pendant le développement du code. Leur nom changera.

<br />


<h2>Langages et outils utilisés</h2>

- <b>Visual Studio Code</b> 
- <b>Jupyter NoteBook</b>
- <b>Miniconda / Anaconda</b>
- <b>Python 3.10.19</b>

<h2>Environnement de travail utilisés </h2>

- <b>Windows 11</b>
- <b>Un environnement virtuel nommé "geo"</b>

<h2>Comment installer l'environnement virtuel "geo" et les prérequis nécessaires : </h2>


- <b>Dans Anaconda Prompt :</b>
- <b>/conda create -n geo python=3.10</b>
- <b>/conda activate geo</b>
- <b>/conda install geopandas rasterion shapely pandas numpy scikit-learn gdal -c conda-forge</b>
- <b>/code .</b>
